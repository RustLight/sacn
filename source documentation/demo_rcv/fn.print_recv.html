fn print_recv(res: Result<Vec<DMXData>>)

Prints the given output from recv to stdout. Errors are printed using their debug output except for universe terminated which is printed as "Universe x Terminated" where x is the universe. This is to avoid the CID being printed which changes for every test as it is randomly generated in most tests.

Arguments

res: The data to display.
